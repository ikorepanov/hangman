## Установка

Клонируем репозиторий:
```shell
git clone https://github.com/ikorepanov/hangman.git
```

Переходим в папку hangman:
```shell
cd hangman
```

## Использование

Запускаем скрипт:
```shell
python src/hangman/main.py
```

или

```shell
python -m src.hangman.main
```

В дальнейшем - следуем указаниям на экране.
